# Affiliate Program Management + CRO Landing Page

Static landing page draft for Cheil Germany's Affiliate Program Management + CRO service.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Open locally
From this folder, open `index.html` in a browser.

If you want to run a quick local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
